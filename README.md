# Agentic-Multi-Document-Research-and-Q-A-Assistant
Agentic Multi-Document Research and Q&amp;A Assistant workshop under the Build with AI UAE initiative, co-branded by Google for Developers and GeeksforGeeks. Made a basic MVP like Google Notebook llm called CyberNote Pro that analyses uploaded pdf's that can be queried and discussed upon with basic flash card functionality built-in.
